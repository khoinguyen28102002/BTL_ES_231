**_Thiết bị sử dụng_**
1. Kit Arduino ESP32
2. Cảm biến nhiệt độ DHT11
3. Cảm biến ánh sáng LM393
4. Cảm biến chuyển động PIR HC-SR501
5. Module 2 Relay 5V Kích mức Cao/Thấp
6. Led 1W CW ánh sáng trắng
7. Động cơ RF-300CA
8. Mạch chuyển đối USB to UART
